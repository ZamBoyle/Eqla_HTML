Exemple de l'attribut readonly

1. Introduction

Le formulaire ci-dessous est en mode lecture seule. Pour le passer en mode édition, cliquez sur le bouton "Passer en mode édition".

Le mode édition supprime l'attribut readonly sur tous les champs du formulaire.

En mode consultation readonly est mis sur tous les champs du formulaire. Cela permet de ne pas modifier les données du formulaire.
